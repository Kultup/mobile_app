# План розробки Mobile додаток (iOS + Android)

## Технологічний стек

- **Фреймворк:** React Native
- **Мова:** TypeScript
- **Навігація:** React Navigation
- **State Management:** Redux Toolkit / Zustand / React Query
- **HTTP клієнт:** Axios
- **Локальне зберігання:** AsyncStorage / MMKV
- **Push-повідомлення:** @react-native-firebase/messaging
- **Офлайн:** React Query з кешуванням
- **Форми:** React Hook Form
- **UI компоненти:** React Native Paper / NativeBase / UI Kitten
- **Адаптивність (КРИТИЧНО):**
  - `react-native-safe-area-context` - обов'язково для вирізів
  - `react-native-responsive-screen` - адаптивні розміри
  - `react-native-device-info` - визначення пристрою
